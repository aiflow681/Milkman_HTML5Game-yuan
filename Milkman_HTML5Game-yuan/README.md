【游戏名称】牛奶人

【游戏介绍】
这是一款H5原生开发的动作冒险类小游戏。游戏中，玩家控制一个可爱的牛奶人角色，在平台上跳跃前进，收集道具，避免掉下平台，最终完成通关。游戏具有流畅的动画效果和简单的操作方式，适合休闲娱乐。

【玩法说明】
操作：
- 使用屏幕上的虚拟控制器控制牛奶人移动
- 点击"⬅️"按钮向左移动
- 点击"➡️"按钮向右移动
- 点击"⬆️"按钮跳跃
- 也可以使用键盘方向键控制
- 使用左右方向键移动，上方向键跳跃

规则：
- 牛奶人需要在平台上跳跃前进
- 收集沿路的道具获得分数
- 避免从平台边缘掉下，否则游戏结束
- 完成整个路程即可通关
- 游戏中有时间限制，需要在规定时间内完成

目标：
- 收集所有道具获得最高分数
- 完成整个关卡路程
- 在时间限制内完成挑战
- 掌握精准的跳跃技巧

Tips：
- 注意观察平台间的距离，合理规划跳跃
- 收集道具时不要过于冒险，安全第一
- 利用连续跳跃技巧，快速通过简单路段
- 观察平台布局，找出最佳路线
- 掌握跳跃力度，避免跳过头或不够

技术介绍：
- 使用HTML5 Canvas和JavaScript开发
- 物理引擎模拟真实的重力、碰撞和摩擦
- 响应式设计，完美适配移动端和桌面端
- 虚拟控制器专为移动设备优化，提供直观的触摸操作
- 高性能动画系统，实现流畅的角色动作
- 双Canvas渲染技术，提高游戏性能和视觉效果
